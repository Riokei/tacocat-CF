Palindrome Check (aka. Taco Cat)

This program will check if the word you entered is a palindrome, (the same word when reversed). This assignment was written in html5, css, js, with the bootstrap framework.

Instructions

  1. Enter a word to see if its a palindrome.
  2. Click "Is this a palindrome?"
  3. Your results will display if it is or isn't!
